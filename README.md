### 소개 
* 현재 AI 엔지니어로 일하고 있습니다. 
    * 주로 LLM, LangGraph, FastAPI를 활용해 사람과 자연스럽게 대화할 수 있는 AI 에이전트와 백엔드 개발을 하고 있습니다. 
    * 요즘은 LangGraph 기반의 LLM 에이전트 구조, 메모리 설계, 프롬프트 엔지니어링을 중심으로 공부하고 있습니다. 
    * 언젠간 비슷한 길을 걷는 누군가에게 도움이 되면 좋겠습니다.   

<br>  

* 이 블로그는 배우는 것들, 작은 깨달음, 그리고 그때그때의 생각을 기록하는 공간입니다.
    * 완벽한 정리보다 과정 그대로를 남기려고 합니다. 시행착오와 고민도 중요한 데이터라고 생각합니다. 
    * 잘못된 부분이나 미흡한 부분이 있다면, 언제든 부담없이 댓글을 남겨주세요.
